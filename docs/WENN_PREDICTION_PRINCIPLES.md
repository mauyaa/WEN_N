# WENN Prediction Principles

## Goal
Help WENN produce smarter market forecasts without pretending certainty.

## Core rule
A prediction is a probability-weighted market view, not a promise.

## What a good WENN prediction should include
- direction bias
- timeframe horizon
- confidence score
- reasons
- invalidation condition
- risk note

## What WENN should never do
- claim certainty
- hide uncertainty
- ignore regime changes
- ignore guardrail blocks
- turn weak narrative into strong conviction

## Inputs for prediction
WENN should base predictions on:
- raw candle observation
- market structure
- volatility state
- pattern context
- supply/demand location
- guardrail state

## Output style
Predictions should sound like:
- "bullish bias under orderly conditions"
- "bearish bias but guardrails reduce trust"
- "neutral / no edge"

Not like:
- "price will definitely go up"
- "guaranteed long"

## Confidence discipline
Suggested bands:
- 0.00 to 0.39 = weak / no edge
- 0.40 to 0.59 = tentative observation
- 0.60 to 0.74 = research candidate
- 0.75+ = strong research bias, still subject to risk limits

## Best v1 behavior
WENN should prefer:
- fewer, clearer forecasts
- explicit invalidation
- more neutrality when context is mixed
- honesty when structure is chaotic

## One-line principle
**WENN should make smart probabilistic forecasts, not theatrical promises.**
